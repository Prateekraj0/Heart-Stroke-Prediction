## 📖 Overview

Heart Stroke Prediction is an end-to-end Machine Learning application that predicts the likelihood of heart disease based on a patient's medical and health-related information.

This project was developed as part of my Data Science learning journey, where I explored the complete machine learning workflow—from data preprocessing and feature engineering to model evaluation and deployment. Multiple classification algorithms were trained and compared to identify the most suitable model for prediction.

The final solution is deployed using Streamlit, providing a simple and interactive interface for users to assess heart disease risk in real time.

## 📋 Input Features

The model uses the following patient health parameters:

| Feature | Description |
|----------|------------|
| Age | Age of the patient |
| Sex | Gender |
| Chest Pain Type | Type of chest pain experienced |
| Resting Blood Pressure | Resting blood pressure (mm Hg) |
| Cholesterol | Serum cholesterol level (mg/dL) |
| Fasting Blood Sugar | Blood sugar level indicator |
| Resting ECG | Electrocardiogram results |
| Max Heart Rate | Maximum heart rate achieved |
| Exercise-Induced Angina | Exercise-related chest pain |
| Oldpeak | ST depression induced by exercise |
| ST Slope | Slope of the peak exercise ST segment |

## 🧠 Machine Learning Models Compared

The following machine learning algorithms were trained and evaluated:

- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)
- Decision Tree
- Naive Bayes

### 🏆 Best Model

After evaluating all models, **Logistic Regression** achieved the highest overall performance and was selected as the final model for deployment.
